## Goals
- Measure discovery success and itinerary completion.
- Track engagement with trivia/history.
- Identify common drop-off points.

## Core Events
- Search initiated
- Filter applied
- Results displayed
- Event detail requested
- Itinerary built
- Routing guidance delivered
- Resource info requested

## KPIs
- Time to first useful answer
- Itinerary completion rate
- Click-through to event links
- Re-query rate due to missing info

## Segmentation
- Weekend vs yearlong queries
- Accessibility and family-friendly filters
- New vs returning sessions
